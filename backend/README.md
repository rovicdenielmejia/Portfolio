# Backend Editor Setup

This project is a Next.js backend editor for your website, deployable on Vercel and using Neon database. It supports page editing, image uploads, blog management, and user roles.

## Features
- Page editing
- Image uploads
- Blog management
- User roles

## Tech Stack
- Next.js (App Router, TypeScript, Tailwind CSS)
- Neon Database
- Vercel Deployment

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Configure Neon database connection in `.env`.

## Deployment
- Deploy to Vercel for production.

- Uses `@app/*` for imports.
